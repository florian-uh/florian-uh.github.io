faq-accordion

Exemple de mise en page concernant diverses questions pouvant être posées, avec un minimum de Javascript pour masquer ou faire apparaitre les questions.
Le principal but de cet exercice est de commencer à se familiariser avec le D.O.M, comprendre son fonctionnement.
La difficulté principale est de faire en sorte que le contenu de la question s'affiche ou se masque en fonction de l'état du bouton, sans pour autant deplacer l'ensemble du contenu.
Il faut egalement que lorsque qu'une question est ouverte, toutes les autres se ferment, cela doit se faire sans à-coups pour cela on peut faire un mix entre JavaScript et CSS.
